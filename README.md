# 🧱 活在肥宣 · Survive-HFUT

更适合合工大学生的生活指南

## 简介

[![Star Chart](https://starchart.cc/Survive-HFUT/survive-hfut.github.io.svg)](https://gitHub.com/Survive-HFUT/survive-hfut.github.io)

这是一份面向合工大学生的生活指南，最初主要针对**宣城校区**编写，目前也在持续完善**合肥校区**相关内容。

我们希望通过更系统、更易检索的方式，把校园学习、生活、办事与常见问题汇总成一个可靠的入口。

### 你能在这里找到

- 新生必看与常见问答
- 校园设施、交通与生活服务
- 学习与考试相关流程
- 组织、活动与校园文化
- 联系方式与实用黄页
- ……

> [!warning]
> 这**不是**合肥工业大学的官方文档。此文档目前主要由学生维护，所有内容均不代表校方官方观点和意见。因更新滞后或其他问题，部分内容可能与实际情况存在出入，请以学校发布的相关管理办法为准

## 参与方式

### 我只是想了解或补充内容

- 直接阅读文档内容，反馈缺失与错误
- 参与方式详见[关于](./docs/about/contribute.md)

### 我想参与开发或构建

确保已安装：

- Git 或 GitHub Desktop
- [nodejs](https://nodejs.org/)
- [pnpm](https://pnpm.io/zh/)

#### 安装依赖

```sh
pnpm install
```

#### 运行即时预览

```sh
pnpm run docs:dev
```

#### 构建文档

```sh
pnpm run docs:build
```
