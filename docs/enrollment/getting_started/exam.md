# 入学考试

几乎所有新生都需要参加英语分级考试

## 英语分级考试

入学后会安排大学英语分级考试，一般是穿插在军训期间。题型与全国大学英语四六级题型相似，包括听力、阅读、词汇与语法等[^1]

### 影响

根据考试结果，将学生编入三个级别，各级别参加四级考试的时间和教学内容会有所区别：[^1]

- 中班和快班的同学可以在当年年底参加英语四级考试，慢班的同学则需要在第二年年底参加英语四级考试
- 快班需上三个学期（一年半）的英语课，其余班级需上四个学期（两学年）的英语课
