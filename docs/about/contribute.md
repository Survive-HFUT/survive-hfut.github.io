# 参与进来

- 如果你打算写一些具体的文档内容的话，可以先阅读[编写规范](./standard.md)
- 如果只是想反馈问题的话，别忘了描述清楚具体情况（如页面链接、相关截图等），以便我们更快地定位和解决问题
- 如果以上对你来说都有点难度的话，也不妨多多向别人宣传安利这个网站，让更多人加入进来一起完善这个项目吧！😋
- 你也可以查看我们的 [待完善事项 (TODO)](./todo.md)，看看有哪些你可以帮忙的地方。

## 提交反馈 & 代理提交

- 发现错别字？
- 内容过时了？
- 有更好的建议？
- 有一腔热血但是不太熟悉复杂的 GitHub 操作和开发流程？

## Pull request

1. 阅读[开发教程](./development.md)
2. [fork](https://github.com/Survive-HFUT/survive-hfut.github.io/fork) 后，进行修改
3. 发起 Merge 请求
