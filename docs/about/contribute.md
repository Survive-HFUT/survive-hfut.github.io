- [腾讯文档收集表](https://docs.qq.com/form/page/DTmd5anpGbmJHUGd6)（匿名反馈）
- [交流群](https://qm.qq.com/cgi-bin/qm/qr?_wv=1027&k=hnhYa-TdwN9v_2f4wXmayC1V0vcdNUEx&authKey=pZs4xDQrK0eFcl2oKMtHB2E9O%2FqXX8SA%2FNNELsIR9t6kgC0YKqxVTrdl9t%2FpI6nO&noverify=0&group_code=812229258)（支持上传截图或 Word 文档）
- [GitHub Issue](https://github.com/Survive-HFUT/survive-hfut.github.io/issues/new)
- [GitHub Discussion](https://github.com/orgs/Survive-HFUT/discussions/new?category=%E5%8F%8D%E9%A6%88)

:::warning 提交反馈时请注明

- 页面链接
- 具体问题描述
- 相关截图（如有）
  :::

### 方式二：在线编辑

适合修改错别字、补充少量内容。

1. 在网站底部点击「在 GitHub 上编辑此页」按钮
2. 点击右上角的 ✏️ 铅笔图标，直接修改内容
3. 滚动到底部，填写修改说明，点击「Commit changes」
4. 系统会自动提示你创建 Pull Request，等待审核合并即可

:::info 提示
这种方式不需要安装任何软件，适合小改动。
:::

## "正在发生"事项快捷提交

如果你想为首页的"正在发生"列表补充新事项（如校运会、考试周等），现在可以通过 GitHub Issue 表单快速提交，机器人会自动为你生成 Pull Request。

1. 访问 [GitHub Issue 提交页面](https://github.com/Survive-HFUT/survive-hfut.github.io/issues/new/choose)
2. 选择 **"📅 新增/更新'正在发生'事项"** 模板
3. 按要求填写事项名称、日期、适用校区等信息
4. 提交后，机器人会自动解析表单并生成 Pull Request，等待管理员审核通过即可上线
