# PWA 支持

本站支持 PWA（Progressive Web App），你可以将其安装到设备上独立使用。

安装后可以获得：

- **独立窗口** — 打开时没有浏览器地址栏和标签页
- **离线浏览** — 已访问过的页面在无网络时也可查看
- **桌面图标** — 从主屏直接打开

## Android

使用 Chrome 浏览器打开本站：

1. 点击地址栏右侧的 `⋮` 菜单
2. 选择 **安装应用**（或 **添加到主屏幕**）
3. 点击 **安装** 确认

安装后可在桌面和应用列表中找到入口。

## iOS（iPhone / iPad）

使用 Safari 浏览器打开本站：

1. 点击底部分享按钮 `□↑`
2. 向下滑动并选择 **添加到主屏幕**
3. 点击右上角 **添加** 确认

安装后可在主屏找到图标，支持独立窗口和离线访问。

## 桌面端

### Edge

1. 点击地址栏右侧的安装图标 `⊕`
2. 弹出窗口点击 **安装** 确认
3. 安装后可从开始菜单或桌面直接启动

也可通过菜单 `设置及其他 → 应用 → 将此站点作为应用安装` 完成。

### Chrome

1. 点击地址栏右侧的安装图标 `⊕`
2. 弹出窗口点击 **安装** 确认

如果未出现安装图标，可通过菜单 `更多工具 → 创建快捷方式…`，然后勾选 **在窗口中打开**。

## 更新

Service Worker 会在每次访问时自动检查更新并在后台加载最新内容。
