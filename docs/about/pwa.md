# PWA 支持

本站支持 PWA（Progressive Web App）。将其安装至设备后，可作为独立客户端使用。

安装后可获得以下体验：

- 独立窗口：无浏览器地址栏和标签页，界面更纯净
- 离线浏览：在无网络环境下，仍可查看已访问过的页面
- 桌面入口：可从主屏幕或应用列表直接打开

## Android

建议使用 Chrome 或系统自带的支持 PWA 的浏览器打开本站：

1. 点击地址栏右侧的 `⋮` 菜单
2. 选择 <mark>安装应用</mark>（或 <mark>添加到主屏幕</mark>）
3. 在弹出的窗口中点击 <mark>安装</mark> 确认

## iOS / iPadOS

使用 Safari 浏览器打开本站：

1. 点击底部的分享按钮（带有向上箭头的方框）
2. 向下滑动并选择 <mark>添加到主屏幕</mark>
3. 点击右上角 <mark>添加</mark> 确认

## 桌面端

### Microsoft Edge

1. 点击地址栏右侧的安装图标 `⊕`
2. 在弹出窗口中点击 <mark>安装</mark> 确认

若未出现安装图标，也可通过右上角菜单 `…` 中的 `应用 → 将此站点作为应用安装` 进行安装。

### Google Chrome

1. 点击地址栏右侧的安装图标 `⊕`
2. 在弹出窗口中点击 <mark>安装</mark> 确认

若未出现安装图标，可通过右上角菜单 `…` 中的 `更多工具 → 创建快捷方式…`，并勾选 <mark>在窗口中打开</mark>。

## 更新机制

本站的 Service Worker 会在每次访问时自动检查更新并在后台加载最新内容，下次打开时即可生效。
