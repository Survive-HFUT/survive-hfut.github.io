## “正在发生”事项快捷提交

为首页的[正在发生](../../ongoing)列表补充新事项（如校运会、考试周等），可以通过 GitHub Issue 表单快速提交，机器人会自动生成 Pull Request：

1. 访问 [GitHub Issue 提交页面](https://github.com/Survive-HFUT/survive-hfut.github.io/issues/new?template=ongoing_event.yml)
2. 填写事项名称、日期、适用校区等信息
3. 提交后，机器人会自动解析表单并生成 Pull Request，等待仓库维护者审核通过即可上线
