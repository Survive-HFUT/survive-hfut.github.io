# 合肥

合肥市，简称“合”，古称庐州、庐阳，是安徽省辖地级市、省会、特大城市，也是合肥都市圈核心城市。合肥位于中国华东地区、安徽省中部、长江三角洲西翼，环抱巢湖，全市总面积 11445 平方千米。这里属亚热带季风性湿润气候，四季分明，地貌以丘陵岗地为主，兼有低山残丘与低洼平原。合肥地域是中华文明的重要发祥地之一，因东淝河与南淝河均发源于此而得名。[^1]
