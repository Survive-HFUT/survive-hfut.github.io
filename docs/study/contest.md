# 竞赛

在大学中各位会遇到各种各样的竞赛，比如大创，互联网+等等。

排除有些需要通过实验室参加的比赛，绝大多数比赛都是要通过自己组队报名参加的。

无论你打算保研，考研，还是本科就业，打个比赛都是有用的，方便你面试时的谈论。

## 赛事列表

:::tip

除了 AB 类（见下）以外的竞赛，根据[**创新创业竞赛及学术竞赛加分规则**](exemption#创新创业竞赛及学术竞赛加分规则)，不同院可能针对一些不在 AB 类赛事中的比赛制定保研加分政策。如计算机与信息学院 [关于 2026 届推荐优秀应届本科毕业生免试攻读研究生相关工作的通告](https://ci.hfut.edu.cn/info/1063/15120.htm)，**这些加分政策通常只适用于发布这些加分政策的学院，且每年都可能有变化**，具体加分事宜可以参考自己所在学院的文件以及询问辅导员

:::
