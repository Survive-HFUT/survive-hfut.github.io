# 奖学金

<ToDo content="本页需要补充更多社会捐赠奖学金、校级奖学金的详细信息" />

学校为学生提供多种奖学金，包括国家级奖学金、校级奖学金和社会捐赠奖学金。

:::info

不同奖学金的申报时间、条件各不相同。具体要求以当年通知为准

:::

## 国家奖学金

### 申请条件

- 品学兼优，无违规记录
- 学业成绩排名位于专业前 10%，或排名虽未进入前 10%，但达到前 30%，且在其他方面表现突出（如获得国家级或省级竞赛奖励）[^3]

### 奖励标准

- 每生 8000 元/年

### 评选时间

通常在每年秋季学期进行，具体时间以[学生工作办公室通知](http://xgzx.hfut.edu.cn/)为准。

## 国家励志奖学金

### 申请条件

- 道德品质良好，无违规记录
- 学业成绩排名位于专业前 25%
- 家庭经济困难，生活俭朴[^1]

:::warning

同一学年内，同一学生不能同时获得国家奖学金和国家励志奖学金

:::

### 奖励标准

- 每生 5000 元/年

### 评选时间

通常在每年秋季学期进行。

## 校级奖学金

学校会按学年评选校级奖学金，常见的是**优秀学生奖学金**[^11]。

:::details 优秀学生奖学金

- 评选对象：国家普通高校招生计划正式招收的全日制本科生、预科班学生
- 奖励标准：一等奖 4000 元/年（3%）、二等奖 2000 元/年（6%）、三等奖 1000 元/年（14%）
- 申请条件：完成本专业教学计划基本内容，必修课无不及格
- 评选依据：学生素质综合测评成绩和名次
- 其他要求：申请国家奖学金、国家励志奖学金的学生需先参加当年校优秀学生奖学金评选

:::

学校还会在不同年份设置其他校设奖助学金，比如斛兵学子系列奖学金等。具体对象、条件和时间以当年通知为准。

## 社会捐赠奖学金

学校接收社会捐赠，设立多种企业和个人捐赠奖学金。常见的有金汇奖学金[^10]、安徽得奇环保科创奖[^12]、龙旗奖学金[^4]、三七互娱大学生赋能奖学金[^5]、天源迪科奖学金[^6]、芯碁微装奖学金[^7]、青苗奖学金和优教奖教金[^8]、院长奖学金（博士）[^9]、晨露助学金、国元证券奖助学金、王老吉未来菁英奖学金、泰有引力·海陵新能源奖学金、比亚迪奖学（教）金[^10]。具体名单、条件、金额和时间看当年通知就行。

### 其他学院及校区特定奖学金

具体奖学金信息由各学院发布，可咨询所在学院学生工作办公室。要看细则的话，直接翻学院通知就行。

## 申报流程

:::info

一般流程如下，具体以当年通知为准：

1. 学院/学工办发布通知，明确申报条件和截止时间
2. 学生准备材料，向辅导员或相关部门提交申请
3. 学院评审小组进行初审和评比
4. 学校公示获奖学生名单
5. 奖学金发放

:::
