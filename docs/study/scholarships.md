# 奖学金

学校为学生提供多种奖学金，包括经济奖励的国家级奖学金、校级自设奖学金和社会捐赠奖学金。

:::info

不同奖学金的申报时间、条件各不相同。具体要求以当年通知为准

:::

## 国家奖学金

### 申请条件

- 热爱祖国，拥护中国共产党领导，品学兼优
- 遵纪守法，无违规记录
- 学业成绩排名位于专业前 10%，或学业成绩排名虽未进入前 10%，但达到前 30%，且在其他方面表现特别优异（如获得国家级或省级竞赛奖励）

### 奖励标准

- 每生 8000 元/年

### 评选时间

通常在每年秋季学期进行，具体时间以[学生工作办公室通知](http://xgzx.hfut.edu.cn/)为准。

## 国家励志奖学金

### 申请条件

- 热爱祖国，拥护中国共产党领导，道德品质优良
- 自觉遵守宪法和法律，遵守学校规章制度，无违规记录
- 学业成绩排名位于专业前 25%
- **家庭经济困难**，生活俭朴

:::warning

同一学年内，同一学生不能同时获得国家奖学金和国家励志奖学金

:::

### 奖励标准

- 每生 5000 元/年

### 评选时间

通常在每年秋季学期进行。

## 学校自设奖学金

学校设有多种自设奖学金，包括斛兵学子系列奖学金等。这类奖学金通常面向特定条件的学生（如家庭经济困难、学业成绩优异等）。

:::details 斛兵学子系列奖学金

学校设置的自设奖学金，包括但不限于"暖冬助学金""桐城美好乡村助学金"等。各学院会在符合条件时组织评选，具体要求和时间以学院通知为准

:::

## 社会捐赠奖学金

学校接收社会捐赠，设立多种企业和个人捐赠奖学金，如金汇奖学金、安徽得奇环保科创奖等。这些奖学金的申报条件、金额和时间因捐赠方而异。

### 常见社会捐赠奖学金

:::tabs key:campus

== 合肥校区

具体奖学金信息以当年通知为准，可咨询所在学院学生工作办公室

== 宣城校区

- **金汇奖学金**：面向计算机与信息系等家庭经济困难学生，标准通常为 4000 元/生
- 其他捐赠奖学金由各学院或部门公布

:::

## 申报流程

:::info

一般流程如下，具体以当年通知为准：

1. 学院/学工办发布通知，明确申报条件和截止时间
2. 学生准备材料，向辅导员或相关部门提交申请
3. 学院评审小组进行初审和评比
4. 学校公示获奖学生名单
5. 奖学金发放

:::

## 常见问题

#### ❔ 可以同时申请国家奖学金和国家励志奖学金吗

**不可以**。同一学年内，同一学生不能同时获得国家奖学金和国家励志奖学金。但可以同时申请国家助学金。

#### ❔ 奖学金金额能用于学费和住宿费吗

可以。奖学金发放后学生可自由支配，用于学费、住宿费或其他生活开支均可。

#### ❔ 学业成绩排名怎么算

通常指综合测评成绩或学业成绩排名。具体计算方法以学校和学院规定为准，可向辅导员或教务部门咨询。

#### ❔ 获奖后有什么要求吗

奖学金获得者需要保持良好的学业表现和品德。若出现违规行为或学业成绩下降显著，学校可能会取消部分获奖资格或要求返还奖学金<Note>需要验证具体规定</Note>。

#### ❔ 社会捐赠奖学金每年都有吗

社会捐赠奖学金因捐赠方的资金情况而定，不同年份和学院的情况可能有差异。具体是否设立和条件以当年学院通知为准。
