# 三下乡

“三下乡”一般指大学生暑期社会实践。它不是一门课程，而是一类在暑期集中开展的社会实践项目，常见形式包括团队立项、专项实践和个人“返家乡”实践

学校团委通常会先发布全校通知，宣城校区再由学生工作办公室/团委结合校区实际另行安排。报名、交材料、结项和认定时，应优先看自己校区、学院（系）和辅导员转发的要求。[^1][^2]

## 参加方式

学校 2026 年通知面向全日制在籍本科生、研究生，实践形式以团队为单位申报。每支团队一般为 6 至 10 人，每名学生原则上只能参加一支团队，实践时长原则上不少于 7 天。[^1]

### 实践类型 [^1][^2]

::::tabs key:campus
== 合肥校区

- “揭榜挂帅”专项实践：围绕地方实际需求选题，团队结合专业优势设计实践方案
- 特色实践路线：围绕“沿着总书记足迹学思想”“弘扬伟大长征精神，走好新时代长征路”等路线设计实践方案
- 暑期社会实践专项行动：包括大国重器、乡村振兴、中华文脉、志愿精神、基层治理、民族团结、就业实习、励志成长等方向
- 团中央、团省委各类专项实践：按对应专项实践通知自行组织申报

== 宣城校区

- “揭榜挂帅”专项实践：围绕地方实际需求选题，团队结合专业优势设计实践方案
- “挺膺担当”主题实践：按学校主题实践内容申报，分为校级资助团队和校级自筹团队
- “寻访红色皖南”“皖南青语”专项实践：围绕红色文化寻访、宣讲等内容开展
- 团中央各类专项实践：按团中央专项实践通知自行组织申报
- 个人实践：一般需向所在系提出申请，并填写对应申请表

::::

:::tip

如果只是想完成社会实践、第二课堂认定或积累经历，可以先关注所在学院（系）、辅导员和团委发布的团队招募。成熟团队通常会提前确定选题、分工、指导教师和材料节点，参与成本会低一些。

:::

## 材料提交

团队实践一般以学院、系或单位为单位统一提交。不同校区、不同组织类型的报送地址和邮箱不一样，应以当年通知为准。

常见材料包括：

- 社会实践团队项目申报书
- 社会实践团队申报汇总表
- 安全责任书
- 结项申报表、结项汇总表
- 实践成果、照片、新闻稿、调研报告等支撑材料
- 评优申报表、推荐汇总表和作品材料

:::warning

- 涉及盖章、指导教师签字、保险、安全责任书、实践打卡和结项材料的要求，建议提前准备
- 材料缺失可能影响立项、结项验收、第二课堂认定或评优推荐

:::

## 第二课堂认定

学校 2026 年通知要求实践团队每日完成“第二课堂成绩单”系统中的安全实践打卡[^1]

宣城校区会对团队进行验收，合格者给予第二课堂成绩单相应星级认定，不合格者不予认定。[^2]

## 结项要求

- 学校立项团队实行“先立项实施、后结项定级”
- 实践结束后，学校会综合实践开展情况和成果情况进行总结验收
- 结项通过后，根据定级给予相应经费支持和第二课堂成绩单“社会实践”模块星级认定[^1][^2]

### 必须准备

学校立项团队结项至少要准备这些材料，具体格式和成果类别以当年附件为准：[^1]

- 实践团队结项申请书
- 团队实践打卡记录
- 与实践主题相关的实践成果。2026 年学校通知要求完成并提交成果（一）至（五）中的 1 项，以及成果（六）至（八）中的 2 项
- 实践照片、宣传报道、调研报告、视频、资政报告、建议方案等支撑材料

换句话说，实践成果不能只交照片或新闻稿。建议从选题阶段就确定调研报告、视频、宣传报道、方案类成果等至少几类材料由谁负责。

### 可选冲刺

如果只是保证结项，通过上述底线材料即可开始准备。若团队想争取较高定级、优秀实践成果、优秀实践团队，或者希望成果更有含金量，可以提前考虑这些方向：

| 方向          | 可以准备什么                                     | 适合什么团队                             |
| ------------- | ------------------------------------------------ | ---------------------------------------- |
| 高质量调研    | 扎实的问卷、访谈、数据分析和 5000 字以上调研报告 | 有明确调研对象、能持续跟进问题的团队     |
| 实践视频      | 5 分钟左右视频，呈现实践过程、问题意识和实际成果 | 有人负责拍摄、剪辑和素材管理的团队       |
| 方案类成果    | 资政报告、建议方案、设计方案、问题解决方案等     | 选题和地方治理、产业需求结合较紧的团队   |
| 文学/宣讲成果 | 微文学作品、宣讲稿、红色故事整理、课程化材料等   | 红色文化、理论宣讲、支教类团队           |
| 媒体报道      | 校级、地市级、省级及以上主流媒体报道             | 有明确活动亮点、能提前联系宣传渠道的团队 |

:::warning

省级及以上媒体报道不是临时投一篇稿就一定能发。想冲省级、国家级平台，应尽早确定选题亮点、图片授权、新闻稿审核流程和投稿渠道，并和指导教师、学院（系）宣传口径保持一致。

:::

:::tip

调研报告、媒体报道、视频和高清过程照片都适合边实践边积累。等到 9 月结项时再补很容易来不及

:::

## 宣城本地项目

如果想报的项目就在宣城，可以关注微信小程序“实践青春 扬帆宣城”，尝试联系有实践需求的单位

与其他类型的项目相比，这类项目离学校近，沟通成本低，但仍然要提前确认实践内容、对接人、地点、时间、食宿、交通、保险、补贴或其他费用
