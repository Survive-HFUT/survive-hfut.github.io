# 选课

## 分类

选修课分为以下几种

### 体育课

算在学业成绩中的，大一和大二由自己选择心仪的体育项目。

### 通识教育选修课（俗称公选课）

这种课从大一下开始自己选，课的种类丰富多彩，需要该该门类选修课一定学分才能顺利毕业，成绩只对申外有用

公选课可分为线下课和慕课，每学期选课数量分别不超过 2 门与 1 门

线下课通俗来说就是需要到教室上的课，这类课大部分都排在晚上，考勤一般相对宽松，期末考大部分开卷或可耍小聪明

慕课类似于网课，大部分课程内容为外校制作，学生需要根据要求在学习通平台上观看视频完成视频任务点、提交作业或讨论（~~大部分就是水评论。只要把别人的话复制下来再发出来就能拿分~~）。有线下的期末考，但是考前通常有题库可以刷题，题库数量略有差异，通常在 200~500 之间

:::tip

- 这类选修课相对专业课来说会简单一点，拿高分也相对容易一些
- 对于选修课的选择与建议，可以参考以下链接
  - 合肥校区
    - [2026 公选课统计表](https://docs.qq.com/sheet/DYmVaaGVoTEZKSHZS)
  - 宣城校区
    - [2024-2025 学年第一学期选修课程调查](https://docs.qq.com/sheet/DS2dxU1VEY1BOTXFx)

:::

### 跨专业选修课

自 2023 级学生起，学生毕业时至少要选修 2 学分<mark>非本专业开设的专业选修课</mark>。学校每学期会组织各学院开设跨专业选修课[^1]

## 公选课要求

- 自 2021 级开始，学生毕业时其通识教育选修课学分中<mark>“四史”课程</mark>学分应不低于 1 学分[^2]
- 自 2023 级开始，学生毕业时<mark>美育类课程</mark>必须修满至少 2 个学分，且美学和艺术史论类、艺术鉴赏和评论类课程至少取得 1 个学分。[^2]
- 学生毕业时通识选修课必须修满至少 10 学分<Note>需要准确来源</Note>
- 以下七大类通识选修课中需修满不少于五大类<Note>需要准确来源</Note>
  - 哲学思想与批判性思维
  - 文史经典及文化传承
  - 科技发展与科学精神
  - 当代中国与社会责任
  - 人文素养与艺术审美
  - 生态环境与可持续发展
  - 国际视野与文明发展

:::tip

你可以在 <mark>教务系统-培养方案完成情况</mark> 中查看已修的学分数量与课程类别

:::

## 选课步骤

通常分为四个阶段（肥区）[^3]/三个阶段（宣区）[^1]

:::tabs key:campus
== 宣城校区

|        | 名称                   | 时间         |
| ------ | ---------------------- | ------------ |
| 第一轮 | 正选                   | 学期倒数一周 |
| 第二轮 | 补选、重修及跨专业选课 | 下学期第一周 |
| 第三轮 | 个人调整选课           | 下学期第一周 |

== 翡翠湖校区

|        | 名称             | 时间           |
| ------ | ---------------- | -------------- |
| 第一轮 | 正选             | 学期倒数第三周 |
| 第二轮 | 正选补选         | 学期倒数第二周 |
| 第三轮 | 重修及跨专业选课 | 学期开学第一周 |
| 第四轮 | 个人调整选课     | 学期开学第二周 |

== 屯溪路校区

与翡翠湖校区一致

:::

### 正选

系统将依据优选范围、课程容量、冲突情况，按照机会均等的原则，对所有的选课申请进行**随机筛选**[^1]

:::tip

第一轮选课并不需要赶时间进入选课系统。若选课人数>开课人数，无论什么时候选都有可能被踢出

:::

### 正选补选

在第一轮选课数据处理完成后，学生上网查看选课结果，对于未选中的课程，可根据选课系统中课程预告的容量余额，在第二轮选课规定的时间内再次上机提交选课申请，但应注意避免与选中课程上课时间的冲突[^3]

对于宣区的同学而言，此轮属于竞争选课，即选课系统不再作非优选类别的数据剔除，**先到先得，选完即止**[^1]

:::tip

在此轮选课中可提前确定好心仪的课程，直接使用课程代码搜索，快人一步

:::

### 重修及跨专业选课

跨年级重修、跨专业选修、学籍异动补修等选课申请可从本轮次开始提交，选课系统不再作非优选类别的数据剔除，先到先得，选完即止[^1][^3]

### 个人调整选课

此轮选课期间，学生可以进行增选、改选、退选课程，选课模式为竞争选课模式，数据是实时动态的，先选先上且选中的课程即刻出现在课表上，改选时要防止退选后无课可选的情况发生[^1]
