# 考试

考试通常在结课后的一周至一个月内进行，具体时间和考试教室可在 教务系统→我的考试 中或 [教务办公室·考试信息（宣城校区）](https://xcjwb.hfut.edu.cn/1178/list.htm) 查询

## 类型

### 机考

通常是[思政类课程](#思政类课程)和[慕课](./electives#慕课)会采取此类方案

大部分考试要求在指定的教室进行，考试期间一般只能携带一台电子设备进入考场（通常是携带手机和一个充电宝）。形势与政策、大学生劳动教育等课程的考试和一些其他考试（如 实验室安全准入）的要求相对较为宽松，可以在截止前任意选择时间和地点（包括宿舍或校外）进行考试

<mark>大部分线上考试并不意味着可以拍照搜题。</mark>以较常使用的学习通平台为例，考试期间多次切屏或累计超过一定时间上限将计为违规并自动交卷。除此之外，根据主考老师在考试中的设置，学习通还能启用其他防作弊功能，如题目乱序、选项乱序、录屏监控识别、人脸识别、禁止复制粘贴、截图检测

### 手写

在纸质试卷或答题纸（卡）上作答

大部分考试仍手改阅卷，英语常使用机改阅卷，目前也有一些课程在逐渐尝试转变为机改阅卷。机改阅卷与高考相似，扫描答题卡后自动根据题目位置分块，随后发送给不同的教师进行批改，教师端一般无法直接查看到整卷和考生个人信息

## 题库

大部分机考课程在考试前的一段时间会开放题库供学生刷题练习，与考试所用的题库可能一致或有重合

有动手能力的同学可以通过下载[notnotype/xxt](https://github.com/notnotype/xxt)或安装[超星作业题库提取器](https://scriptcat.org/zh-CN/script-show-page/3514)插件爬取学习通上指定课程的所有题目，方便刷题自测

## 斩杀线

> 凡期末考试卷面成绩**低于 50 分（不含 50 分）**，即认定该课程不及格，总评成绩直接按期末考试卷面成绩计入。[^2]

<mark>若期末考试卷面分数低于此分数，该课程直接被认定为不及格。</mark>换句话说，这是确保这门课程能不挂科的期末考最低分数

从 2025-2026 第二学期开始，斩杀线有所变化，从原有的 45 分提升至 50 分[^1]

## 作弊处理

:::danger

**诚信考试 杜绝作弊**

:::

从 2025 年开始，作弊不再取消学位证[^3]<Note>需要更多来源信息</Note>。一说此举是为了整顿考试作弊。<Note>在原有的政策规定下一旦被发现作弊就直接取消学位证，这意味着为了作弊后想要获得学位证只能重新高考，可见后果之严重，部分老师、学院可能因此放宽监考或处罚要求；而新规实行后监考老师、学院和校方均不再会有此类的顾虑，因此监考和处罚力度可能加大</Note>
