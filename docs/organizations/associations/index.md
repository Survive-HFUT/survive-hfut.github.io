# 社团

学校社团招新统一于百团大战，一般在十月初

:::info

部分社团可能因为各种原因失传、解散，如宣区的天文社（2023 年停止纳新）

期待之后能有新同学重新创办，延续学长学姐们的心血和付出，为社团繁荣注入新的活力

:::
