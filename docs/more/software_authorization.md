# 软件正版化

学校为师生提供了多种正版软件授权，包括 Windows 操作系统、Office 办公套件、MATLAB 等。[^1]

:::tip 使用前提
所有软件的下载和激活**必须**在**校园网**环境下进行（或通过 VPN 接入校园网）。
:::

## 平台访问

- **官方网站**：[http://ms.hfut.edu.cn/](http://ms.hfut.edu.cn/)

## 可用软件

### 1. 微软产品 (Microsoft)

- **操作系统**：Windows 11 / 10 / 8.1 / 8 (均为 64 位)
- **Office 套件**：
  - **Windows**: Office 2024 / 2021 / 2019 / 2016 LTSC
  - **Mac**: Office for Mac 2024 / 2021 / 2019 / 2016 LTSC
- **Visio**: 2024 / 2021 / 2019 / 2016 (均为 64 位)

### 2. 学科及工具软件

- **MATLAB**: 提供 2023b 及历史版本（支持 Windows, Mac, Linux）。
- **Foxit PDF Editor**: 福昕 PDF 编辑器专业版。
- **WPS Office**: 已由教育专业版升级为 **WPS 365 教育旗舰版**。包含 WPS AI、多人在线协作、500 人在线会议等功能。[^2]

## 激活方法

### Windows, Office 及 Visio (客户端激活)

目前学校主要采用 **GP 激活客户端** 进行激活。

1. 访问 [正版化平台首页](http://ms.hfut.edu.cn/)，在首页找到并下载 **“GP 激活客户端”**（GP Activation Client）。
2. 安装客户端并运行，使用**信息门户（学工号）**账号密码登录。
3. 在列表中选择需要激活的产品，点击 **“立即激活”**。
4. 等待提示激活成功。

:::warning 激活次数限制
每个账号通常有 **10 次** 激活 Windows 和 Office 的尝试机会。如果提示超过次数，可能需要联系信息中心。
:::

### MATLAB (账号激活)

MATLAB 采用 MathWorks 账户绑定授权，无需通过 GP 客户端。

1. 使用学校提供的**校内邮箱**（学生通常为 `<学号>@mail.hfut.edu.cn`）注册或登录 MathWorks 账号。
2. 注册时身份选择 “Teacher” 或 “Student”。
3. 登录后即可关联学校的 Site License 并下载安装激活。

### WPS 365 教育旗舰版 (账号关联)

WPS 365 采用组织架构授权，无需通过 GP 客户端，权益直接绑定个人账号。

1. 打开 WPS 客户端，点击“登录”，选择“其他方式登录”或“第三方账号登录”。
2. 找到并选择“合肥工业大学”或通过**统一身份认证（CAS）**账号登录。
3. 登录成功并成功加入组织后，个人账号将自动解锁旗舰版权益（含 WPS AI、无时长限制会议、超大云存储等）。

## 其他教育优惠 (edu 邮箱)

除了校内直接提供的授权外，学生还可以通过学校分配的 **edu 邮箱**（后缀为 `@mail.hfut.edu.cn`）申请多种商业软件的教育版或优惠。详见 [猜你想问§校园邮箱](../enrollment/qa.md#校园邮箱)。

### 1. JetBrains 系列工具

包含 IntelliJ IDEA, PyCharm, WebStorm, CLion 等开发工具。

- **获取地址**：[JetBrains Student License](https://www.jetbrains.com/shop/eform/students)
- **申请方式**：使用学校邮箱注册并申请。通过验证后可免费使用 JetBrains 旗下的专业版软件。授权有效期通常为一年，到期后可续期。

### 2. Autodesk 系列

包含 AutoCAD, Maya, 3ds Max 等建模和设计软件。

- **获取地址**：[Autodesk Education Community](https://www.autodesk.com.cn/education/edu-software/overview)
- **申请方式**：注册账号并选择学生身份，通过教育邮箱验证。部分情况下可能需要上传学信网验证码或学生证照片。

### 3. GitHub Student Developer Pack

包含多种开发工具、云服务额度（如 Azure, DigitalOcean）以及 Canva Pro 等。

- **获取地址**：[GitHub Education](https://education.github.com/pack)

## 常见问题与维护 [^3]

### 1. 激活失败 (错误代码 0x80080005)

如果遇到此错误，通常是因为系统服务损坏。建议使用 **Office Tool Plus** 等第三方工具重置“软件保护”服务（Software Protection）。

### 2. 清理旧版密钥

如果重装系统或 Office 后无法激活，可以尝试使用 Office 安装目录下的 `ospp.vbs` 脚本手动清除旧的激活信息。
