# 冰糖

冰糖是合肥工业大学于 2014 年 5 月 8 日正式上线的官方 IP 形象，其名称取自校内文化古迹"斛兵塘"的谐音

形象设计中融入了主教红服饰与碧绿瞳孔等校园标志性元素，每年本科生录取通知书中的手幅上的二次元形象正是冰糖
