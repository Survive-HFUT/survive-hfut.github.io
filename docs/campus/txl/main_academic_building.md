# 主教学楼

![主教学楼](./main_academic_building.png)

屯溪路校区的标志性建筑，始建于 1958 年，为苏联式建筑风格，是当年安徽省最高的建筑，如今已被列为合肥市文物保护单位和安徽省文物保护单位。[^1]
