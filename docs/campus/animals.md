# 小动物

校园内有许多小动物（猫、狗、蛇、松鼠、刺猬、黑天鹅等）

宣城校区里的猫猫狗狗挺喜欢偷外卖吃（特别是烤鸭或炸鸡），因此务必尽快取餐，~~避免你的外卖遭受攻击~~

## 流浪猫流浪狗治理计划

根据校方发布的[《关于加强校园流浪猫治理的通知》](https://bwzx.hfut.edu.cn/info/1003/3551.htm)，近期可能对校内的流浪猫或者流浪狗进行一定的治理，有意愿收养流浪猫或流浪狗的同学可以联系下方的小动物保护组织
