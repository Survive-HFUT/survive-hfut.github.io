# 智能制造技术研究院

智能制造技术研究院是由教育部、工信部、安徽省政府支持，合肥市政府与合肥工业大学共建的主要开展科技成果转化及产业化、高科技企业培育、产教融合人才培养的新型研发机构。[^1]

## 历史[^1]

::: timeline 2014
智能制造技术研究院成立
:::

::: timeline 2016
被列为省级“全创改”试点单位
:::

::: timeline 2017
入选合肥综合性国家科学中心
:::

::: timeline 2018
被认定为安徽省首批新型研发机构
:::

::: timeline 2023
被认定为安徽省首批高水平新型研发机构
:::
