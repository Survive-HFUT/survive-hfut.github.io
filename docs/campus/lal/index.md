# 六安路校区

六安路校区位于安徽省合肥市庐阳区六安路 269 号，是四大校区之一，也称北校区。

历史上，该校区系原安徽工学院校址。安徽工学院曾在“文化大革命”期间与合肥工业大学合并，后于 1978 年经国务院批准在原址恢复办学。1997 年 1 月，安徽工学院与合肥工业大学再次合并，组建新的合肥工业大学，并于同年 6 月挂牌成立合肥工业大学六安路校区。[^1]

现在的市政主干道纵贯校园将校园一分为三，校园因地处城市中心而没有任何空间扩展的可能，虽然校区有自成体系的建设，但因校区土地面积太小而难有作为。[^2]

目前，该校区已暂停招生，主要为教职工宿舍和离退休教职工活动中心，但校区内仍保留有[六安路校区综合管理办公室](https://lalxq.hfut.edu.cn/main.htm)等管理机构。同时，校区内拥有合肥市重要历史文化遗迹“娘娘池”。
