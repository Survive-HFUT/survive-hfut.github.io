# 故障报修

如果宿舍内或公共区域的设施出现了故障，可以通过以下方式进行报修：

:::tabs key:campus

== 合肥校区
有以下三种报修方式：

- [信息门户](https://one.hfut.edu.cn/)→选择“智能报修系统”
- 微信公众号“合肥工业大学信息化”→“便捷服务”→“智能报修”
- 网络报修咨询电话`0551-62901178`

== 宣城校区

扫描宿舍门后二维码，或是直接进入[智慧后勤](http://xcfw.hfut.edu.cn/school/index.html#/reload?role=S&t=1696839059992)，使用统一身份认证平台登录后，在“智能报修”中选择“我要报修”，根据提示填写相关信息提交即可。通常在 1~2 个工作日内就会有维修人员确认，小的问题一般在一周内即可完成维修

维修不收取维修费和材料费

:::
