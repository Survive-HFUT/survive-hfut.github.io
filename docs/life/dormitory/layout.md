# 布局

== 屯溪路校区

== 翡翠湖校区

翡翠湖校区目前共有 13 栋学生公寓，均采用先进的电子门禁，由楼宇、公共门厅及内庭院组合而成。学校新改造的四人间宿舍为上床下桌，配有挂式空调、独立卫生间和阳台[^1]

楼宇内设有饮水机、自助洗衣机、浴室、售货机等生活设施，还有一站式服务学生社区，不出楼门就可以进行自习、小组讨论等活动[^1]

== 宣城校区

宿舍楼内通常有浴室、洗衣机、饮水机等公共设施，部分宿舍楼内还有自习室和公共休息区

:::
