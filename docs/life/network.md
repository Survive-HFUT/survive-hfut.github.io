# 校园网

校园网是指学院或大学校园内在各种建筑物之间互连的网络，属于园区网络中的一种。在本指南中，其广义上可指代合肥工业大学校园中的所有网络的总和，包括但不限于接入校园的中国教育和科研计算机网（`CERNET`）、教育无线漫游（`EDUROAM`）、中国移动（`CHINA MOBILE`）、中国电信（`CHINA TELECOM`）和校内局域网等

而对于在校学生，我们常说的校园网一般指的是合工大官方的公用网络，用户通过学生公寓有线（`172.16.*`）或校园无线（HFUT-WiFi：`10.*`; HFUT-EDU:：`114.213.*` ）接入校园局域网、中国教育和科研计算机网（`CERNET`，连接到中国科学技术大学）出口、中国电信（`CHINA TELECOM`）出口和中国移动（`CHINA MOBILE`）出口等网络，除校内局域网和 IPv6 网络以外，通常需要登录由`Drcom`提供的计费认证系统，才能连接到互联网

其他的一些网络，如中国移动宽带、中国电信宽带和各大运营商的移动通信网络等，由于它们不是学校官方建设运营的网络，所以一般不称之为校园网，而以“[运营商网络](#运营商网络)”代之[^3]

:::tip

使用部分校内服务不一定得连校园网，可以使用[合肥工业大学 WEBVPN 系统](https://webvpn.hfut.edu.cn)，即使不在学校也能访问部分校园网内部资源

:::

## 套餐费用 & 网速

::::tabs key:campus

== 合肥校区

对于本科在校学生，2023 年 5 月 25 日以后，校园网分为 4 种套餐：[^3]

| 资费/元 |    速率/Mbps（MB/s）     | 在线终端数量/台 |
| :-----: | :----------------------: | :-------------: |
|    0    |        10 (1.25)         |        2        |
|   10    |        100 (12.5)        |        2        |
|   20    |         200 (25)         |        3        |
|   40    | 不限，实际为 1000（125） |        3        |

:::tip

原 15 元/不限速/2 台在线终端的套餐，在新规实施后并不受影响，依旧按照原来的资费进行缴纳。这使得这个“绝版”套餐在新下具有很高的性价比，已经开通此套餐的同学且用且珍惜[^3]

:::

更改校园网套餐需携带相关证件到综合服务楼负一楼的网络中心进行办理

== 宣城校区

- 每日限额 20G，每个月有 100GB 的免费额度，超出后按 0.5 元/GB 计费，总上限为 300GB；法定节假日和寒暑假免费[^1]，但免费期间可能因为用的人多而导致网速下降
- 使用[慧新易校 APP](../life/app#慧新易校)用校园卡内余额进行充值
- 正常网速可以跑到 200Mbps（25 MB/s） 以上，在一些信号好的教室可能达到 300~400Mbps（37.5~50 MB/s）或更高
- 每个账号只允许在一个设备登录；当新的设备登录后，其他设备将会被挤下线

## 连接方法

::::tabs key:campus

== 合肥校区

校园网络基于 Web 页面认证方式，无需下载客户端。接入网络后，会自动弹出认证页面。也可以手动输入认证页面地址：`172.16.200.11`/`172.16.200.12`/`192.168.4.1`进行认证。如未进行认证，客户端可访问校内相关资源，但不能访问互联网资源。[^4]

== 宣城校区

1. 插入网线或连接`hfut-wlan`/`hfut_wlan` WiFi
2. 连接成功后会自动打开认证界面，若未打开可手动打开[宽带认证系统（宿舍区域、教学楼区域）](http://172.18.3.3) / [宽带认证系统（实验楼、图书馆区域）](http://172.18.2.2)或随便打开一个网页（都会自动跳转到登录界面）
3. 登录
   - 账号为学号（`20xxxxxxxx`）
   - 默认密码为身份证号后 6 位（最后一位是字母的，则为字母前面的 6 位数字）
4. 再次访问上述宽带认证系统，可以查询到本次登录校园网的已使用时间、已使用流量、余额信息。
   - 由于技术原因，实际使用流量为显示的已使用流量的两倍[^2]

:::tip

你也可以安装[哆点校园](https://doctorcom.com/duodian/)实现自动登录和流量查询等功能。

:::

校园网通常不支持多个设备使用同一账号同时登录，不过也存在一些例外情况：

- [宽带认证系统（宿舍区域、教学楼区域）](http://172.18.3.3)和[宽带认证系统（实验楼、图书馆区域）](http://172.18.2.2)似乎相互独立，同一账号可以分别登录。
  - 例如，在图书馆通过[实验楼、图书馆区域认证系统](http://172.18.2.2)登录后，仍可在宿舍通过[宿舍区域、教学楼区域认证系统](http://172.18.3.3)登录同一账号。两端计时、计费相互独立，利用这个特点理论上可将单个账号每月免费流量从 100GB 提高到 200GB。
  - 这两类区域之间暂未观察到设备隔离，因此两台设备可以直接通信。
- 部分由校区信息化建设与发展中心单独开通的组织机构公用账号（如实验室、学生组织账号）也可以多设备同时登录，但通常会限制登录 IP 段，仅允许在对应物理位置的网口接入使用。

若有多设备上网的需求，你可以参考下方的[教程](#共享网络)

:::info

目前的校园网貌似存在一些 Bug：

- 当在一栋教学楼连接并登录到校园网后，走到另外一栋教学楼后连接校园网时不会弹出登录界面，但也没有网络连接

解决方法：

- 换一个设备登录
- 忘记网络后重新加入
- 手动更改当前设备的 MAC 地址来重新登录校园网

:::

## 共享网络

### 笔记本热点

1. 使用**笔记本电脑**先连接校园网并登录（无线和有线皆可）
2. 打开 设置>网络与 Internet>移动热点
3. 点击编辑，修改 WiFi 名称和密码
4. 打开 与其他设备共享我的 Internet 连接
5. 在手机上连接第三步设置的 WiFi

### 路由器

- 共用一台路由器，且使用路由模式，能且只能登录一个账号，费用由最后登录账号结算。[^1]
- 使用集线器，或将路由器以桥接方式使用，个人需各自输入账号，费用分开单独结算。[^1]

## 技术细节（宣区）

- 当你的连入校园网时，IP 地址应在 IPv4 的 20 位块专用网络之内，也就是 `172.16.0.0`–`172.31.255.255`
- 和肥区的校园网不同的是，宣区的校园网<mark>暂未配置客户端隔离功能</mark>，这意味着你开放于校园网的端口可以被校园网内其他设备访问
  - 因此你可以借助此特性在校园网内进行局域网游戏联机（如 [Minecraft](https://minecraft.net/)）或搭建局域网文件传输服务（如 [LocalSend](https://localsend.org/)）等
  - 同理，如果你在校园网内搭建了某些服务（如 Web 服务器、SSH 服务器、代理等），也可能会被校园网内的其他人访问到，因此请注意安全设置，避免被攻击或隐私泄露
- 校园网一般不开启 IPv6 功能，开启的时候据称是在测试
- 学校配布的 AP 为 Wifi6，但 5Ghz 频段的最大频宽仅有 80MHz，这意味着最大连接速度为 1201Mbps
- 断网是由门户自动强制下线学生账户，部分由校区信息化建设与发展中心单独开通的组织机构公用账号不受此限制，但此类账号会限制登录的 IP 段，即仅允许在对应物理位置使用
- 自动登录校园网可以参考 GitHub 仓库 [oriwika/HFUTXC_autologin](https://github.com/oriwika/HFUTXC_autologin)
- 其他细节可以参考以下两个公告：
  - [《合肥工业大学宣城校区校园网出口带宽服务采购项目（BX22000071）采购公告》](https://zb.hfut.edu.cn/provider/#/publish/20L5M80L6LDHEWSE)
  - [《合肥工业大学宣城校区校园网出口带宽服务采购项目（BX22000071）成交结果公告》](https://zb.hfut.edu.cn/provider/#/publish/20L63FCEP1WZE3UY)
