## 参保缴费 💳

想在学校看病省钱，参保是第一步哦！

- 省内同学推荐使用 **“安徽医保公共服务”** 微信小程序。
- 省外同学推荐使用 **“皖事通”** APP。

### 参保登记怎么做？🤔

1. 打开“安徽医保公共服务”微信小程序或“皖事通”APP，登录 **“安徽 e 保”**。
2. 在地方专区记得选择 **“宣城市”**。
3. 如果是第一次用，可能需要先激活电子医保凭证，跟着提示点几下就好。
4. 找到“参保缴费”栏目，点击“城乡居民参保登记”，根据实际情况选择是否代办。
5. 点击“学校基本信息” -> 学校名称选择 **“合肥工业大学宣城校区”** -> 正确填写手机号码 -> 选择参保身份为 **“大学生”** -> 上传身份证照片等。
6. 仔细检查一遍，没问题就点击“提交”啦！

:::info 小贴士：关于“重复参保”的处理

如果你发现系统提醒“此人在 xx 市存在参保信息，不能重复参保”，别慌！这说明你在家里那边已经交过医保了。

- 你需要按原参保地的规定办理“停保”或“暂停参保”，然后再按上面的流程在宣城办登记。
- 如果是省内医保，办完停保后，原有的待遇通常能继续享受到 2024-12-31。
- 安徽省内的同学可以尝试在“安徽医保公共服务”小程序里选择原籍地，找到“城乡居民暂停参保”来操作。也可以直接拨打原籍地医保中心的电话。

:::

### 接下来是缴费环节 💰

#### 微信小程序渠道

1. 在“安徽医保公共服务”小程序里选好“宣城市”。
2. 进入“宣城专区”，依次点击 -> “城乡居民（税务系统）缴费” -> “允许”，然后跟着税务局的小程序提示操作即可。

#### 皖事通渠道

1. 登录“皖事通”APP，定位在“宣城市”。
2. 在首页搜“医保缴费专区”进去，或者直接点首页的图标。
3. 找到“城乡居民医保缴费”版块，点击“点击缴费”。
4. 进入服务后，点“选择人员” -> “添加成员”，填好身份证号和姓名保存。
5. 选好本次缴费的参保人、险种和年度，点“查询”。
6. 确认信息无误后，点击“缴费”就大功告成啦！

## 校外门诊报销 🏥

如果在校外看了门诊，符合条件的医药费是可以报销的，大家记得留意哦！

:::tip 报销的小规则

1. **什么时候报？** 学校通常 **每学期组织一次** 集中报销（一般在 **5 月** 和 **11 月** 左右），涵盖上一阶段产生的医药费。
2. **谁来办理？** 原则上要本人到现场。如果实在去不了，记得写一份亲笔签名的 **委托书** 请同学代办。
:::

### 记得带齐这些材料哦 📝

办理报销的时候，千万别忘了带上这几样，不然就白跑一趟啦：

1. **本人学生证**
2. **校区医院转诊单**（急诊除外，平时记得保留好）
3. **定点医院门诊发票**（如果单次费用 > 500 元，记得带上收费明细）
4. **就诊病历记录**
5. **学校那张农行卡**（用来接收报销款项）

### 报销额度与周期

- **能报多少？** 按照学校规定，每个保险年度的限额是 **2000 元**。
- **发票有效期**：
  - **春季报销（5 月）**：主要报前一年 11 月到今年 5 月产生的费用。
  - **秋季报销（11 月）**：主要报今年 5 月到 11 月产生的费用。

:::info 温馨提示
**具体的办理日期、地点及发票期限，大家一定要以 [宣城校区官网通知公告栏](http://xc.hfut.edu.cn/1955/list.htm) 发布的最新通知为准哦！**
:::
