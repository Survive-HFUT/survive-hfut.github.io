# 软件正版化 & 教育优惠

学校为师生提供了多种正版软件授权，包括 Windows 操作系统、Office 办公套件、MATLAB 等

:::tip 使用前提
所有软件的下载和激活**必须**在**校园网**环境下进行（或通过 VPN 接入校园网）
:::

## 可用软件 & 权益

- 操作系统：
  - Windows 11 / 10 / 8.1 / 8 (均为 64 位)
- Office 套件：
  - Windows: Office 2024 / 2021 / 2019 / 2016 LTSC
  - Mac: Office for Mac 2024 / 2021 / 2019 / 2016 LTSC
- Visio: 2024 / 2021 / 2019 / 2016 (均为 64 位)
- MATLAB: 提供 2023b 及历史版本（支持 Windows, Mac, Linux）
- Foxit PDF Editor: 福昕 PDF 编辑器专业版
- WPS Office: 已由教育专业版升级为 **WPS 365 教育旗舰版**。包含 WPS AI、多人在线协作、500 人在线会议等功能。[^2]
  - _该权益可能已过期，请以实际可用情况为准_

## 激活方法

### Windows, Office 及 Visio

1. 访问[合肥工业大学正版软件管理与服务平台](http://ms.hfut.edu.cn/)，在首页找到并下载 **“GP 激活客户端”**（GP Activation Client）
2. 安装客户端并运行，使用信息门户账号密码登录
3. 在列表中选择需要激活的产品，点击“立即激活”
4. 等待提示激活成功

### MATLAB

MATLAB 采用 MathWorks 账户绑定授权

1. 使用学校提供的**校内邮箱**（学生通常为 `<学号>@mail.hfut.edu.cn`）注册或登录 MathWorks 账号
2. 注册时身份选择 “Teacher” 或 “Student”
3. 登录后即可关联学校的 Site License 并下载安装激活

### WPS 365 教育旗舰版 (账号关联)

:::warning

由于学校采购周期等原因，该权益可能已经过期，请自行登录并测试可用性

:::

WPS 365 采用组织架构授权，无需通过 GP 客户端，权益直接绑定个人账号

1. 打开 WPS 客户端，点击“登录”，选择“其他方式登录”或“第三方账号登录”
2. 找到并选择“合肥工业大学”或通过**统一身份认证（CAS）**账号登录
3. 登录成功并成功加入组织后，个人账号将自动解锁旗舰版权益（含 WPS AI、无时长限制会议、超大云存储等）

## 其他教育优惠 (edu 邮箱)

除了校内直接提供的授权外，学生还可以通过学校分配的 **edu 邮箱**（后缀为 `@mail.hfut.edu.cn`）申请多种商业软件的教育版或优惠

### JetBrains 系列工具

包含 IntelliJ IDEA, PyCharm, WebStorm, CLion 等开发工具

在[JetBrains Student License](https://www.jetbrains.com/shop/eform/students)使用学校邮箱注册并申请。通过验证后可免费使用 JetBrains 旗下的专业版软件。授权有效期通常为一年，到期后需手动续期

### Autodesk 系列

包含 AutoCAD, Maya, 3ds Max 等建模和设计软件

在[Autodesk Education Community](https://www.autodesk.com.cn/education/edu-software/overview)注册账号并选择学生身份，通过教育邮箱验证。部分情况下可能需要上传学信网验证码或学生证照片

### GitHub Student Developer Pack

包含多种开发工具、云服务额度（如 Azure, DigitalOcean）以及 Copilot Pro 等。更多权益和注册方法详见[GitHub Education](https://education.github.com/pack)
