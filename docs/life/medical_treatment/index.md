# 医疗

一些较小的疾病可以直接前往校区医院就诊；对于较为严重的疾病，建议直接前往市内医院就诊

## 外请专家坐诊（宣城校区）

校区医院聘请校外专家定期坐诊[^1]：

- 妇科：每周二、周四下午 2:00-5:00
- 中医科：每周二、周四下午 2:00-5:00
- 放射科：每周五下午 2:00-5:00

预约电话：`0563-3831120`，坐诊地点：校区医院二楼专家诊室
