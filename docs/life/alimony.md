# 生活费

因人而异

| 月消费    | 评价                                                   |
| --------- | ------------------------------------------------------ |
| 600       | 能活                                                   |
| 1000~1500 | 正常水平，三餐能吃饱，偶尔出去搓一顿，月末还可能有结余 |
| 2000+     | 富哥，顺带能包养一个对象                               |
| 五位数+   | ~~这是真的富哥，建议包养我~~                           |

如果家庭条件困难，请联系导员申请贫困生认定/助学金贷款，也可以兼职做家教（可以找学长学姐要群），两小时能赚 150~300。

:::tip

- 开学初或学期初消费可能较大（买书和必要的日用品），可能造成一定经济负担
- 显示器、座椅等大件物品如果不是急需的话可以等次年毕业季收大四学长学姐的二手货[^1]

:::
