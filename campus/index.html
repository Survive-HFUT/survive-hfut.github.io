                                            # 主要建筑

                                            # 图书馆

                                            宣城校区的唯一一个图书馆，有阅览座位 3300 余席、休闲座位 360 席、电子阅览室座位 144 席、研讨间 20 个。

                                            # 开放时间

                                            功能时间段
                                            图书馆开放7:30 - 22:15
                                            图刊借阅8:00 - 22:00
                                            电子阅览室8:00 - 22:00
                                            研讨间8:10 - 22:00
                                            报刊阅览8:00 - 22:00

                                            # 楼层

                                            校史馆在第十层。

                                            # 风雨操场

                                            标准四百米跑道，中间是足球场。

                                            看台下有校区警务站、舞蹈房。

                                            入口处有一台自动体外除颤器（ AED ）设备。

                                            周围有篮球场、网球场和排球场。
